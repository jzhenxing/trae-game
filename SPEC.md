# 像素风机甲对战游戏 - 产品需求文档

## 1. 产品概述

一款复古像素风格的双人机甲对战小游戏，两个玩家操控各自机甲进行战斗。游戏融合经典街机格斗游戏的操作手感与现代视觉效果，呈现硬核科幻与像素艺术的完美结合。

### 核心定位
- 目标用户：喜欢复古街机游戏和科幻题材的玩家
- 核心价值：快速上手、即时对战的本地双人对战体验
- 情感诉求：怀旧感、紧张感、成就感

## 2. 核心功能

### 2.1 机甲角色系统

#### 机甲 A - 突击型（STRIKER）
- 外观：红色涂装、紧凑身形、强调攻击性
- 特性：高攻击力、中等防御、中等移动速度
- 技能：快速连击、冲刺攻击

#### 机甲 B - 防御型（TANKER）
- 外观：蓝色涂装、厚重装甲、强调稳重感
- 特性：中等攻击力、高防御、低移动速度
- 技能：重击、格挡姿态

### 2.2 战斗系统

| 动作 | 玩家1（P1） | 玩家2（P2） | 效果 |
|------|-----------|-----------|------|
| 移动左 | A | ← | 向左移动 |
| 移动右 | D | → | 向右移动 |
| 跳跃 | W | ↑ | 向上跳跃 |
| 攻击 | J | Numpad 1 | 普通攻击 |
| 防御 | K | Numpad 2 | 减少受到的伤害 |
| 特殊技能 | L | Numpad 3 | 释放强力技能（消耗能量） |

### 2.3 数值系统

- **生命值 (HP)**：100点
- **能量值 (SP)**：100点，自动恢复
- **普通攻击伤害**：10点
- **特殊技能伤害**：25点
- **防御减伤**：50%
- **能量消耗**：特殊技能消耗30点能量

### 2.4 胜负判定

- 当任意一方生命值归零时，比赛结束
- 胜利方显示胜利动画和结算界面
- 支持重新开始功能

## 3. 界面设计

### 3.1 游戏场景

- **战斗区域**：800x400像素的矩形战场
- **地面**：像素风格金属地板，带有科幻网格纹理
- **背景**：深邃太空场景，带有星云和远景机甲剪影
- **氛围**：霓虹灯光效果、粒子特效

### 3.2 UI 元素

- **生命条**：顶部横条，红色渐变，实时更新
- **能量条**：生命条下方，蓝色/黄色渐变
- **角色名称**：生命条上方显示 "P1" / "P2"
- **操作提示**：底部显示当前玩家操作按键

### 3.3 视觉风格

- **主色调**：深空黑 (#0a0a12) + 霓虹蓝 (#00d4ff) + 警报红 (#ff3366)
- **像素密度**：8-bit 复古像素风格
- **动画效果**：
  - 攻击闪烁效果
  - 受伤抖动效果
  - 防御光盾效果
  - 跳跃重力感
- **字体**：像素风格等宽字体

## 5. 技术实现

### 5.1 渲染技术
- Canvas 2D 渲染
- RequestAnimationFrame 循环
- 像素-perfect 渲染（禁用抗锯齿）

### 5.2 物理系统
- 简化的重力模拟
- 碰撞检测（矩形碰撞）
- 移动边界限制

### 5.3 动画系统
- 帧动画控制器
- 状态机管理（待机、移动、攻击、防御、受伤、死亡）
- 特效粒子系统

## 6. 素材方案

### 6.1 机甲精灵
使用 Canvas 绘制像素风格机甲：
- 待机动画：2帧
- 移动动画：4帧
- 攻击动画：3帧
- 防御动画：2帧
- 受伤动画：2帧

### 6.2 场景素材
- 地面纹理：程序化生成网格图案
- 背景：星空 + 远景剪影
- 特效：粒子、光效、震动

### 6.3 UI 素材
- 像素风格按钮
- 渐变进度条
- 像素字体渲染

## 7. 性能目标

- 目标帧率：60 FPS
- 响应延迟：< 16ms
- 兼容浏览器：Chrome, Firefox, Safari, Edge
