# 霓虹战机 - NEON STRIKER 产品需求文档

## 1. 产品概述

一款复古霓虹风格的单人格斗射击游戏（Vertical Shooter），玩家操控战机在无限纵版空间中躲避敌方弹幕并击落敌机。游戏采用 80 年代街机美学，融合赛博朋克霓虹视觉与像素艺术，带来紧张刺激的弹幕躲避体验。

## 2. 核心功能

### 2.1 游戏角色

| 角色 | 描述 |
|------|------|
| 玩家战机 | 玩家控制的霓虹战机，支持移动射击 |
| 普通敌机 | 从屏幕顶部随机生成，水平飞行并射击 |
| BOSS | 每隔一定分数出现的高血量精英敌机 |

### 2.2 功能模块

1. **开始界面** - 标题动画、开始按钮、最高分显示
2. **游戏主界面** - 战机移动、弹幕射击、敌机生成、碰撞检测
3. **分数系统** - 击毁敌机得分、连击奖励
4. **HP 系统** - 玩家拥有 3 条生命，被击中减少一条
5. **游戏结束界面** - 显示最终得分、重新开始按钮

## 3. 核心流程

```
开始界面
    ↓ [点击开始]
游戏主循环
    ├── 玩家移动 (WASD/方向键)
    ├── 玩家射击 (空格键自动射击)
    ├── 敌机生成 (从顶部随机位置出现)
    ├── 子弹移动 & 碰撞检测
    ├── 分数更新
    └── HP 变化
    ↓ [生命归零]
游戏结束界面
    ↓ [重新开始] → 游戏主循环
```

## 4. 视觉设计

### 4.1 设计风格

- **主题**：复古霓虹街机 + 赛博朋克
- **背景**：深空黑色 (#0a0a12)，带有扫描线 CRT 效果
- **主色调**：青色 (#00f0ff)、品红色 (#ff00aa)、明黄色 (#ffee00)
- **字体**：Press Start 2P（像素风格）
- **特效**：霓虹辉光、粒子爆炸、屏幕震动

### 4.2 UI 元素

| 元素 | 样式 |
|------|------|
| 标题文字 | 青色霓虹辉光，脉冲呼吸动画 |
| 玩家战机 | 青蓝色像素飞机，带引擎尾焰 |
| 敌机 | 品红/橙色像素飞机 |
| 子弹 | 黄色/青色光弹，带拖尾 |
| 爆炸 | 粒子向外扩散，颜色渐变 |
| HP 条 | 绿色渐变，低于 30% 变红闪烁 |
| 分数 | 右上角，青色霓虹数字 |

### 4.3 响应式适配

- 移动端：支持触摸虚拟摇杆移动，自动射击
- 桌面端：WASD/方向键移动，空格键射击
- 画布自适应屏幕宽度，最大宽度 480px，居中显示

## 5. 游戏平衡

| 参数 | 数值 |
|------|------|
| 玩家移动速度 | 6px/帧 |
| 玩家射击间隔 | 150ms |
| 敌机生成间隔 | 初始 1500ms，每 500 分减少 100ms，最低 600ms |
| 敌机移动速度 | 2-4px/帧（随机） |
| 敌机射击间隔 | 2000ms |
| BOSS 出现间隔 | 每 3000 分触发一次 |
| 初始生命 | 3 条 |
